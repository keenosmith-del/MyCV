# 🌐 Personal Portfolio Website

A fully responsive personal portfolio website designed to showcase my background, skills, and web development projects in a clean and interactive way.

> 💡 *This project is continuously evolving as I refine my skills and expand my experience in web development.*

## 📌 Overview

This project is a modern, responsive web portfolio that highlights my journey as a developer. It brings together my education, technical skills, and previous work into a single, easy-to-navigate experience. The design focuses on simplicity, accessibility, and smooth navigation across different screen sizes.

## ✨ Features

* **Responsive Design**
  The layout adapts seamlessly across various screen sizes, providing an optimized experience for desktops and tablets.

* **Dynamic Navigation**
  A menu-based navigation system allows users to quickly jump between different sections of the page.

* **Project Showcase**
  Displays a curated selection of my previous web development projects with visual and interactive elements.

* **Personal Background & Skills**
  Includes sections outlining my education, technical skill set, and professional background.

* **Social & Contact Links**
  Easy access to my social media profiles and contact information.

* **Adaptive Scrolling Experience**

  * Horizontal scrolling for larger screens to create a unique browsing experience
  * Vertical scrolling for smaller screens to maintain usability

## ⚙️ Functionality

* Interactive buttons and navigation links are implemented
* Core UI/UX elements are functional and responsive
* Some features are intentionally minimal, focusing on presentation rather than complex backend functionality

## 📱 Mobile Experience

While the site is responsive, it is **not yet fully optimized for mobile devices**. Future improvements will focus on enhancing usability and layout for smaller screens.

## 🚀 Future Improvements

* Full mobile optimization
* Enhanced interactivity and functionality
* Possible integration with backend services (e.g., contact form handling)
* Performance and accessibility improvements

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📎 Purpose

This portfolio serves as a representation of my current skills and growth as a web developer. It is both a showcase of completed work and a foundation for future enhancements.
